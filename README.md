# Promedio semanal del clima en Python

Este repositorio contiene dos implementaciones para calcular el promedio semanal del clima:

- **Programación Tradicional:** `tradicional.py`
- **Programación Orientada a Objetos (POO):** `poo.py`

## Cómo ejecutar
Desde la terminal, dentro de la carpeta del proyecto:

```bash
python tradicional.py
python poo.py
